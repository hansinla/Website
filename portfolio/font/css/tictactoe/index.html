TicTacToe Help

Use the dropdown-menu button to change the play mode between Manual mode, Computer plays X, and Computer Plays O. Changing mode resets the game.

Click the Score button to reveal game statistics.

Click Reset to terminate current game and start again. The current game will not be evaluated in the statistics.

Click Help to reveal this Help dialog. Click Close or X to dismiss.
